# jwp-shopping-cart 기능목록

## 상품 목록 페이지 연동
- [ ] 상품은 아래와 같은 필드를 가지고 있다.
  - ID(long)
  - Name(String)
  - Image(String)
  - Price(int)

- [ ] 컨트롤러는 아래와 같은 요청을 처리한다.
  - [ ] '/' 경로로 get 요청이 들어올 경우, 상품 목록 페이지를 반환한다.

- [ ] 상품서비스는 아래와 같은 기능을 한다.
  - [ ] 전체 상품 목록을 DB에서 가져온다.
  - [ ] 전체 상품 목록을 컨트롤러에 보낸다.
